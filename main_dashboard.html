Завантажити дані
TSV/CSV з Google Sheets або Planfix. Колонки: Номер, Назва, Статус, Група, Постановник, Виконавець, Дата виконання, Дата завершення, Дата створення, FTR призначення, FTR відповідь.
⊞
Перетягніть файл або натисніть
.tsv, .csv, .txt — експорт з Google Sheets
В Google Sheets: Файл → Завантажити → TSV (.tsv)
Файл не передається в мережу — обробляється локально.
◫
Оберіть папку з TSV/CSV/TXT файлами
Можна зберігати щоденні експорти в окремій папці
Після імпорту дані запам'ятовуються в localStorage. Для безпосереднього читання папки browser повинен отримати дозвіл.
Google Sheets (публічний доступ):
https://docs.google.com/spreadsheets/d/ID/export?format=tsv

Локальний HTTP-сервер:
http://localhost:8080/tasks.tsv

Примітка: браузер блокує прямі file://-шляхи. Для локальних файлів — вкладка Файл.
⟲
Завантажте історію статусів
Формат: Номер, дата переходу, статус
Очікується файл з колонками: Номер, дата переходу в статус, статус. Формат дати: MM/DD/YYYY H:mm.
